## 02_02_img01: scanner_slit_scanning.svg — [판정: ✅]

**SVG 코드 검증:**
- **화살표 방향**:
    - 마스크: `<line x1="400" ... x2="460" ... marker-end="url(#arrowRed)"/>` (x2 > x1) → **오른쪽(정방향)** 확인.
    - 웨이퍼: `<line x1="170" ... x2="110" ... marker-end="url(#arrowRed)"/>` (x2 < x1) → **왼쪽(역방향)** 확인.
    - Marker: `refX="9"`, `path d="... L10,3.5 ..."` → 팁이 10에 있고 9에서 연결되므로 **정확한 끝점 연결** 확인.
- **축척 (4:1)**:
    - 마스크 쪽 슬릿 폭: `290 - 210 = 80`.
    - 웨이퍼 쪽 슬릿 폭: `260 - 240 = 20`.
    - 비율: 80:20 = **4:1** → 스펙 일치.
- **텍스트 정렬**:
    - 중앙 정렬 `x="250", text-anchor="middle"` 일관 적용.
    - 타이틀(y=35), 마스크(y=80), 슬릿(y=135), 렌즈(y=252), 웨이퍼(y=432) → **수직 정렬 일관됨**.
- **색상**:
    - Slit: `#FFD700` (Gold) → 스펙 일치.
    - Mask: `#444444` → 스펙 일치.
    - Wafer: `#C0C0C0` → 스펙 일치.
    - Lens: `#87CEEB` (SkyBlue) → 스펙 일치.
- **라벨 정확성**:
    - "마스크", "속도 v", "슬릿 (좁은 노광 영역)", "투영 렌즈", "4:1 축소", "웨이퍼", "속도 v/4" → **스펙 텍스트 전수 일치**.
- **요소 겹침**:
    - 마스크 패턴(y=72~98)과 슬릿 빔 시작점(y=110) 분리 → 겹침 없음.
    - 웨이퍼 패턴(y=422~448)과 슬릿 빔 끝점(y=410) 분리 → 겹침 없음.

**판정 사유:** SVG 코드 상의 좌표, 색상, 텍스트 내용이 스펙과 완벽히 일치하며, 광학적 축소 비율(4:1)과 이동 방향이 물리적으로 올바르게 구현됨.

## 02_02_img03: dual_stage_concept.png — [판정: ✅]

**PNG 시각 검증:**
- **구조 완전성**: 두 개의 스테이지(Measure/Expose)가 병렬로 배치됨.
- **프로세스 묘사**:
    - Measure(좌측): 파란색 빔(센서)으로 웨이퍼 스캔.
    - Expose(우측): 주황색 빔(노광)으로 레티클 통과 후 웨이퍼 전사.
    - Swap: 중앙의 순환 화살표로 스테이지 교체 표현.
- **타임라인**: 하단에 Stage A/B의 병렬 작업(Measure N+1 / Expose N) 및 Swap 타이밍이 명확히 도식화됨.
- **텍스트 정확성**: "Sensors", "Reticle", "Wafer N", "Wafer N+1" 등 라벨 오류 없음.

**판정 사유:** 복잡한 듀얼 스테이지 개념을 직관적인 다이어그램으로 명확히 표현하였으며, 타임라인과 스테이지 동작이 논리적으로 일치함.

## 02_02_img04: illumination_shapes.svg — [판정: ✅]

**SVG 코드 검증:**
- **구조 완전성**: 4개 패널(Conventional, Annular, Dipole, Quadrupole) 모두 존재.
- **도형 형상 검증 (Pupil Plane)**:
    - **Conventional**: `r="30"` 원형 (중앙) → 확인.
    - **Annular**: `r="50"` 원에서 `r="28"` 원 제외 (Ring 형태) → 확인.
    - **Dipole**: `cy="52"`, `cy="128"` (중심 90 기준 상하 대칭) 타원 2개 → **수직형(Vertical)** 확인.
    - **Quadrupole**: (62.5, 60), (122.5, 60), (62.5, 120), (122.5, 120) → 중심(92.5, 90) 기준 대각선 대칭 → **사극형** 확인.
- **색상**:
    - 조명 영역: `#FFD700` (Gold) → 스펙 일치.
    - 배경: `#1a1a2e` (Dark Blue) → 스펙 일치.
- **텍스트 일관성**: 각 패널 하단에 Title(Bold, 13px) + Description(Regular, 10px) 구조 통일.
- **뷰박스**: `viewBox="0 0 800 250"` → 4분할 레이아웃에 적합.

**판정 사유:** 조명계의 4가지 주요 모드가 기하학적으로 정확하게 구현되었으며, 각 모드의 특징(Dipole 수직, Quadrupole 대각선 등)이 스펙과 일치함.
